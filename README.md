# homework2
Вектор хранит только ненулевые элементы в отсортированном односвязном списке.
Операции сложения и вычитания выполняются за O(nnz), где nnz — число ненулевых элементов.
Память освобождается в деструкторе, используется RAII.
Порядок индексов контролируется checkListOrder()
main.cpp нужен для локального тестирования реализации класса.
CLion работает через CMake, поэтому CMakeLists, а не Makefile
